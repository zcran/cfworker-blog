## 找出第 N 个二进制字符串中的第 K 位


给你两个正整数 n 和 k，二进制字符串  Sn 的形成规则如下：

S1 = "0"
当 i > 1 时，Si = Si-1 + "1" + reverse(invert(Si-1))

其中 + 表示串联操作，reverse(x) 返回反转 x 后得到的字符串，而 invert(x) 则会翻转 x 中的每一位（0 变为 1，而 1 变为 0）。

例如，符合上述描述的序列的前 4 个字符串依次是：

S1 = "0"
S2 = "011"
S3 = "0111001"
S4 = "011100110110001"

请你返回  Sn 的 第 k 位字符 ，题目数据保证 k 一定在 Sn 长度范围以内。


```
impl Solution {
    /// 公共接口：查找第 n 个二进制字符串中的第 k 位字符
    ///
    /// # 参数
    /// - `n`: 字符串的序号（1-based）
    /// - `k`: 要查找的位置（1-based）
    ///
    /// # 返回
    /// - 该位置的字符（'0' 或 '1'）
    pub fn find_kth_bit(n: i32, k: i32) -> char {
        Self::find_kth_bit_recursive(n, k)
    }

    /// 递归查找第 k 位字符
    ///
    /// 算法核心：利用 Sn 的结构特点直接定位，无需构造整个字符串
    ///
    /// Sn 的结构可以表示为三部分：
    /// [第一部分: S(n-1)] + [中间位: "1"] + [第三部分: invert(reverse(S(n-1)))]
    /// 长度关系：len(Sn) = 2^(n) - 1
    /// 中间位位置：mid = 2^(n-1)
    fn find_kth_bit_recursive(n: i32, k: i32) -> char {
        // 基本情况：Sn 的第一个字符总是 '0'
        // 因为 S1 = "0"，且后续构造中第一部分都是 S(n-1)
        if k == 1 {
            return '0';
        }

        // 计算 Sn 的中间位置（即 "1" 的位置）
        // 1 << (n - 1) 等价于 2^(n-1)
        // 例如：n=3 时，len=7，mid=4
        let mid = 1 << (n - 1);

        // 情况1：k 正好是中间位置
        if k == mid {
            return '1';  // 中间位固定为 '1'
        }
        // 情况2：k 在左半部分（属于 S(n-1)）
        else if k < mid {
            // 直接到 S(n-1) 中查找相同位置
            return Self::find_kth_bit_recursive(n - 1, k);
        }
        // 情况3：k 在右半部分（属于 invert(reverse(S(n-1)))）
        else {
            // 关键：将右半部分的位置映射到左半部分的对称位置
            //
            // 对称关系示例（n=3, S3 = "0111001"）：
            // 位置: 1 2 3 4 5 6 7
            // 字符: 0 1 1 1 0 0 1
            //       | | |   | | |
            // 左半: 0 1 1    右半: 0 0 1
            // 反转并取反: 左半"011" -> 反转"110" -> 取反"001"
            //
            // 对称位置映射：new_k = mid * 2 - k
            // 例如 k=5: new_k = 4*2-5 = 3，对应左半位置3
            // 例如 k=6: new_k = 4*2-6 = 2，对应左半位置2
            // 例如 k=7: new_k = 4*2-7 = 1，对应左半位置1
            let new_k = mid * 2 - k;

            // 先递归找到左半对称位置的原始字符，然后取反
            return Self::invert(Self::find_kth_bit_recursive(n - 1, new_k));
        }
    }

    /// 取反操作：'0' -> '1', '1' -> '0'
    fn invert(bit: char) -> char {
        if bit == '0' {
            '1'
        } else {
            '0'
        }
    }
}
```
