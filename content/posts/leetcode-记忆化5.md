## 整数替换

给定一个正整数 n ，你可以做如下操作：

1.如果 n 是偶数，则用 n / 2替换 n 。
2.如果 n 是奇数，则可以用 n + 1或n - 1替换 n 。

返回 n 变为 1 所需的 最小替换次数 。

```
impl Solution {
    /// 返回将整数 `n` 变为 `1` 所需的最小操作次数。
    ///
    /// 允许的操作：
    /// - 如果 `n` 是偶数，可以除以 2。
    /// - 如果 `n` 是奇数，可以加 1 或减 1。
    ///
    /// # 算法思路
    /// 采用贪心 + 位运算策略：每次根据 `n` 的最低两位决定最佳操作路径，
    /// 从而一次跳过多个步骤（函数式递归实现）。
    ///
    /// 具体规则（`n > 3` 时）：
    /// - 若 `n % 4 == 0`（末两位 `00`）：连续两次除以 2 → 2 步。
    /// - 若 `n % 4 == 1`（末两位 `01`）：先减 1（变成 4k），再两次除以 2 → 3 步。
    /// - 若 `n % 4 == 2`（末两位 `10`）：直接除以 2 → 1 步。
    /// - 若 `n % 4 == 3`（末两位 `11`）：先加 1（变成 4k+4），再两次除以 2 → 3 步。
    ///
    /// 特殊情况：`n == 3` 时最优解为 2 步（3 → 2 → 1）。
    pub fn integer_replacement(n: i32) -> i32 {
        // 内部递归函数，使用 i64 避免中间结果溢出
        fn replace(n: i64) -> i32 {
            if n == 1 {
                0
            } else if n == 3 {
                2
            } else {
                match n & 3 {
                    // 末两位 00：n 能被 4 整除，连续两次除以 2
                    0 => 2 + replace(n >> 2),
                    // 末两位 01：减 1 后变成 4 的倍数，再两次除以 2
                    1 => 3 + replace(n >> 2),
                    // 末两位 10：偶数，直接除以 2
                    2 => 1 + replace(n >> 1),
                    // 末两位 11（且 n > 3）：加 1 后成为 4 的倍数，再两次除以 2
                    _ => 3 + replace((n >> 2) + 1),
                }
            }
        }

        replace(n as i64)
    }
}
```
