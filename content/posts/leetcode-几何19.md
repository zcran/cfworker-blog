## 访问所有点的最小时间

平面上有 n 个点，点的位置用整数坐标表示 points[i] = [xi, yi] 。请你计算访问所有这些点需要的 最小时间（以秒为单位）。

你需要按照下面的规则在平面上移动：

· 每一秒内，你可以：
  · 沿水平方向移动一个单位长度，或者
  · 沿竖直方向移动一个单位长度，或者
  · 跨过对角线移动 sqrt(2) 个单位长度（可以看作在一秒内向水平和竖直方向各移动一个单位长度）。

· 必须按照数组中出现的顺序来访问这些点。
· 在访问某个点时，可以经过该点后面出现的点，但经过的那些点不算作有效访问。


```
impl Solution {
    /// 计算按顺序访问所有点的最短时间（单位：秒）
    /// 每秒可以沿水平、垂直或对角线方向移动一步（切比雪夫距离）
    /// 因此，从点 A 到点 B 所需的最短时间为：
    ///     max(|x1 - x2|, |y1 - y2|)
    pub fn min_time_to_visit_all_points(points: Vec<Vec<i32>>) -> i32 {
        // windows(2) 创建相邻点对的迭代器，如 [p1,p2], [p2,p3], ...
        points.windows(2)
            // 对于每对相邻点，计算切比雪夫距离（即步数）
            .map(|w| {
                let dx = (w[0][0] - w[1][0]).abs(); // x 方向差值绝对值
                let dy = (w[0][1] - w[1][1]).abs(); // y 方向差值绝对值
                dx.max(dy) // 切比雪夫距离 = 两个差值的最大值
            })
            // 累加所有相邻段的时间，得到总时间
            .sum()
    }
}
```
