## 移除最小数对使数组有序 I

给你一个数组 nums，你可以执行以下操作任意次数：

选择 相邻 元素对中 和最小 的一对。如果存在多个这样的对，选择最左边的一个。
用它们的和替换这对元素。
返回将数组变为 非递减 所需的 最小操作次数 。

如果一个数组中每个元素都大于或等于它前一个元素（如果存在的话），则称该数组为非递减。



示例 1：

输入： nums = [5,2,3,1]

输出： 2

解释：

元素对 (3,1) 的和最小，为 4。替换后 nums = [5,2,4]。
元素对 (2,4) 的和为 6。替换后 nums = [5,6]。
数组 nums 在两次操作后变为非递减。

示例 2：

输入： nums = [1,2,2]

输出： 0

解释：

数组 nums 已经是非递减的。


```
impl Solution {
    pub fn minimum_pair_removal(mut nums: Vec<i32>) -> i32 {
        let mut ans = 0;
        while nums.windows(2).any(|p| p[0] > p[1]) {    // windows(2) 创建所有相邻元素对的迭代器。any 检查是否存在逆序对（前 > 后）。只要还有逆序对，就继续循环
            let (new_elem, i) = nums
                .windows(2)       // 生成相邻元素对，如 [(a,b), (b,c), (c,d)]
                .enumerate()      // 为每个对添加索引，(0, (a,b)), (1, (b,c)), (2, (c,d))
                .map(|(i, p)| (p[0] + p[1], i))     // 转换为 (和, 索引)
                .min()      // 找到和最小的相邻对
                .unwrap();      // 因为存在逆序对，所以至少有一个元素
            nums[i] = new_elem;   // 将第i个元素替换为两数之和
            nums.remove(i + 1);   // 移除第i+1个元素
            ans += 1;             // 操作次数加1
        }
        ans   // 返回总共的操作次数
    }
}
```

最小配对移除算法，通过反复合并相邻的逆序对，最终使数组变为非递减序列。

算法目标

将数组通过以下操作变为非递减序列（即对于所有 i，nums[i] <= nums[i+1]）：

找到相邻元素和最小的逆序对（nums[i] > nums[i+1]）
将它们合并为一个元素（值为两数之和）
重复直到没有逆序对
返回操作次数
