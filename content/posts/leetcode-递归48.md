## 汉诺塔问题


在经典汉诺塔问题中，有 3 根柱子及 N 个不同大小的穿孔圆盘，盘子可以滑入任意一根柱子。一开始，所有盘子自上而下按升序依次套在第一根柱子上(即每一个盘子只能放在更大的盘子上面)。移动圆盘时受到以下限制:
(1) 每次只能移动一个盘子;
(2) 盘子只能从柱子顶端滑出移到下一根柱子;
(3) 盘子只能叠在比它大的盘子上。

请编写程序，用栈将所有盘子从第一根柱子移到最后一根柱子。

你需要原地修改栈。



```
impl Solution {
    // 汉诺塔问题：将 A 柱上的所有圆盘移动到 C 柱
    // 参数：
    //   a: 源柱子（起始柱子）
    //   b: 辅助柱子（中转柱子）
    //   c: 目标柱子
    pub fn hanota(a: &mut Vec<i32>, b: &mut Vec<i32>, c: &mut Vec<i32>) {
        // 获取源柱子上的盘子数量
        let n = a.len() as i32;
        // 调用递归函数开始移动盘子
        Self::dfs(n, a, b, c);
    }

    // 递归移动函数
    // 参数：
    //   n: 需要移动的盘子数量
    //   a: 源柱子（起始柱子）
    //   b: 辅助柱子（中转柱子）
    //   c: 目标柱子
    fn dfs(n: i32, a: &mut Vec<i32>, b: &mut Vec<i32>, c: &mut Vec<i32>) {
        // 基本情况（递归出口）：只有一个盘子时
        if n == 1 {
            // 直接将这个盘子从源柱子 a 移动到目标柱子 c
            // pop() 从 a 中取出盘子，push() 放入 c
            c.push(a.pop().unwrap());
            return;
        }

        // 步骤1：将上面 n-1 个盘子从 a 移动到 b（借助 c）
        // 此时：
        //   - 源柱子：a
        //   - 目标柱子：b
        //   - 辅助柱子：c
        Self::dfs(n - 1, a, c, b);

        // 步骤2：将最下面的第 n 个盘子（最大的盘子）从 a 移动到 c
        // 这是唯一可以直接移动的盘子（上面没有其他盘子）
        Self::dfs(1, a, b, c);

        // 步骤3：将 b 上的 n-1 个盘子从 b 移动到 c（借助 a）
        // 此时：
        //   - 源柱子：b
        //   - 目标柱子：c
        //   - 辅助柱子：a
        Self::dfs(n - 1, b, a, c);
    }
}
```
