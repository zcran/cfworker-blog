## 多数元素

给定一个大小为 n 的数组 nums ，返回其中的多数元素。多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。


```
impl Solution {
    /// 寻找多数元素（Boyer-Moore 投票算法）
    ///
    /// # 算法原理
    /// 多数元素出现次数 > ⌊n/2⌋，意味着它比其他所有元素的总和还要多。
    /// 算法通过“抵消”策略：不同元素相互抵消，最后剩下的就是多数元素。
    ///
    /// # 步骤说明
    /// 1. 假设第一个元素是候选多数元素，计数器设为 1
    /// 2. 遍历后续元素：
    ///    - 如果计数器为 0，选择当前元素作为新候选，计数器设为 1
    ///    - 如果当前元素等于候选，计数器 +1
    ///    - 如果当前元素不等于候选，计数器 -1（相互抵消）
    /// 3. 遍历结束后，候选元素即为多数元素
    ///
    /// # 复杂度
    /// - 时间：O(n)，单次遍历
    /// - 空间：O(1)，只使用两个变量
    ///
    /// # 示例
    /// ```
    /// nums = [2, 2, 1, 1, 1, 2, 2]
    /// 过程：2(1) → 2(2) → 2(1) → 2(0) → 1(1) → 1(0) → 2(1)
    /// 结果：2 ✓
    /// ```
    pub fn majority_element(nums: Vec<i32>) -> i32 {
        let mut candidate = 0;  // 当前候选的多数元素
        let mut count = 0;      // 候选元素的“净优势”计数

        for &num in nums.iter() {
            if count == 0 {
                // 没有候选时，当前元素成为新候选
                candidate = num;
                count = 1;
            } else if num == candidate {
                // 相同元素，增加优势
                count += 1;
            } else {
                // 不同元素，相互抵消
                count -= 1;
            }
        }

        candidate
    }
}
```
