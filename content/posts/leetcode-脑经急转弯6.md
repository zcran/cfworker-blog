## 飞机座位分配概率

有 n 位乘客即将登机，飞机正好有 n 个座位。第一位乘客的票丢了，他随便选了一个座位坐下。

剩下的乘客将会：

如果他们自己的座位还空着，就坐到自己的座位上，

当他们自己的座位被占用时，随机选择其他座位
第 n 位乘客坐在自己的座位上的概率是多少？



///


这个问题是经典的“飞机座位分配概率”问题，通常被称为“第100个乘客问题”的推广版本。我来详细解析这个概率问题的结论和推导过程。

---

## 1. 问题重述

- 有 n 个座位，编号 1 到 n
- 有 n 位乘客，乘客 i 的票对应座位 i
- 第 1 位乘客（丢票者）随机选择一个座位坐下
- 后续乘客：
  - 如果自己的座位空着，就坐自己的座位
  - 如果自己的座位被占了，就随机选择其他空座位坐下
- 问：第 n 位乘客（最后一位乘客）坐在自己座位（座位 n）上的概率是多少？

---

## 2. 经典结论

**概率恒为 1/2，对所有 n ≥ 2 成立。**

- n = 1 时，概率为 1（只有一个人，肯定坐自己座位）
- n ≥ 2 时，概率 = 1/2

---

## 3. 直观理解

这个结果有点反直觉——看起来概率似乎应该随着 n 增大而趋近于 0，但实际是稳定的 1/2。

### 3.1 对称性解释

**关键观察**：整个过程只会出现两种情况：

1. 第一位乘客恰好坐到了自己的座位（座位 1）
   → 后续所有人都会坐自己的座位 → 最后一位乘客肯定坐自己座位

2. 第一位乘客坐到了最后一位乘客的座位（座位 n）
   → 最后一位乘客肯定坐不到自己座位

3. 第一位乘客坐到了其他座位 k（2 ≤ k ≤ n-1）
   → 乘客 2, 3, …, k-1 都会坐自己的座位
   → 轮到乘客 k 时，发现座位被占，他成为“新的丢票者”，在座位 1 和座位 {k+1, …, n} 中随机选择
   → 问题递归为规模更小但结构相同的问题

### 3.2 递推关系

设 P(n) 为 n 个乘客时，最后一位乘客坐到自己座位的概率。

第一位乘客的选择：
- 以概率 1/n 选座位 1 → 最后一位成功概率 = 1
- 以概率 1/n 选座位 n → 最后一位成功概率 = 0
- 以概率 (n-2)/n 选座位 k（2 ≤ k ≤ n-1）
  → 乘客 2 到 k-1 都会坐自己座位
  → 乘客 k 面临类似问题，但现在是：
    - 剩余座位：座位 1 和座位 {k+1, …, n}（共 n-k+1 个座位）
    - 乘客 k 相当于新的“丢票者”
    - 剩余乘客数为 n-k+1
  → 这个子问题中，最后一位（原乘客 n）坐自己座位的概率是 P(n-k+1)

因此：
```
P(n) = (1/n)×1 + (1/n)×0 + (1/n) × [P(n-1) + P(n-2) + … + P(2)]
```

但更优雅的推导是：

**对称性论证**：
- 在第一位乘客随机选座后，对于座位 1 和座位 n，它们的“命运”是对称的
- 最终只有两个座位可能决定结果：座位 1 和座位 n
- 在第一个乘客选座后，这两个座位中，必然有一个会被“特殊处理”——要么直接有人坐，要么成为递归的焦点
- 最终，座位 1 和座位 n 被最后一个人（或递归中的关键人物）选中的概率相等
- 因此，最后一位乘客得到自己座位的概率 = 1/2

---

## 4. 数学归纳法证明

**基础**：n = 2 时
- 第 1 人随机选座：
  - 选座位 1（概率 1/2）→ 第 2 人坐座位 2（成功）
  - 选座位 2（概率 1/2）→ 第 2 人坐不到座位 2（失败）
- 概率 = 1/2

**归纳假设**：对所有 m < n，P(m) = 1/2

**归纳步骤**：考虑 n
- 第 1 人选座位 1：概率 1/n，最终成功
- 第 1 人选座位 n：概率 1/n，最终失败
- 第 1 人选座位 k（2 ≤ k ≤ n-1）：概率 (n-2)/n 中每个 k 概率 1/n
  - 乘客 2 到 k-1 坐自己座位
  - 乘客 k 面对：座位 1 + 座位 k+1 到 n（共 n-k+1 个座位）
  - 这是规模为 n-k+1 的子问题（最后一位仍是乘客 n）
  - 由归纳假设，成功概率 = 1/2

因此：
```
P(n) = (1/n)×1 + (1/n)×0 + Σ_{k=2}^{n-1} (1/n) × (1/2)
     = 1/n + (n-2)/n × (1/2)
     = 1/n + (n-2)/(2n)
     = (2 + n - 2) / (2n)
     = n/(2n)
     = 1/2
```

---

## 5. 扩展思考

### 5.1 为什么不是 1/n？
直觉可能认为：最后一位乘客只有当他自己的座位没被前面任何人选时才成功。第 1 人选座位 n 的概率是 1/n，看起来像是答案，但实际是 1/2，因为递归过程中，座位 1 和座位 n 始终处于对称位置。

### 5.2 模拟验证
对于大 n，比如 n = 100，通过计算机模拟也可以验证概率稳定在 0.5 左右。

### 5.3 变体问题
如果第一位乘客不是完全随机，而是以某种概率分布选座，结论会变化。但本题中，随机选座导致了这种对称性。

---

## 6. 总结

| n | 概率 |
|---|------|
| 1 | 1 |
| 2 | 1/2 |
| 3 | 1/2 |
| 4 | 1/2 |
| ... | 1/2 |

**核心要点**：
- 问题简化为座位 1 和座位 n 之间的博弈
- 这两个座位在递归过程中保持对称性
- 最终概率恒为 1/2（n ≥ 2）

这个问题的美妙之处在于，看似复杂的随机过程，最后却得到了一个极其简洁的结果。
