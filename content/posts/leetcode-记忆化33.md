## 到达第 K 级台阶的方案数

给你有一个 非负 整数 k 。有一个无限长度的台阶，最低 一层编号为 0 。

Alice 有一个整数 jump ，一开始值为 0 。Alice 从台阶 1 开始，可以使用 任意 次操作，目标是到达第 k 级台阶。假设 Alice 位于台阶 i ，一次 操作 中，Alice 可以：

向下走一级到 i - 1 ，但该操作 不能 连续使用，如果在台阶第 0 级也不能使用。
向上走到台阶 i + 2jump 处，然后 jump 变为 jump + 1 。

请你返回 Alice 到达台阶 k 处的总方案数。

注意，Alice 可能到达台阶 k 处后，通过一些操作重新回到台阶 k 处，这视为不同的方案。

```
impl Solution {
    /// 计算到达楼梯第 k 级的不同方式数（LeetCode 3154）。
    ///
    /// 数学原理：
    /// - 设向下操作的次数为 `n`（即“后退”的次数），则向上跳的步数总和为 `2^n`。
    /// - 每向下一步会减少一次向上的机会，所以总移动次数为 `2^n - n`，
    ///   但最后一次到达 `k` 后不能再操作，因此实际总步数为 `2^n - n - 1`。
    /// - 在总步数中，有 `down` 次向下操作，其余向上，最终位置为：
    ///   `(向上总步数) - (向下总步数) = (2^n - down) - down = 2^n - 2*down`，
    ///   结合总步数约束 `2^n - n - 1 = 向上次数 + 向下次数 = (2^n - down) + down` 恒成立，
    ///   可推出最终位置 = `2^n - n - 1 - down`，其中 `down` 可取 `0..=n`。
    /// - 因此对于给定的 `n`，可达位置范围为 `[2^n - n - 1, 2^n]`，
    ///   当目标 `k` 落在此区间时，方案数等于从 `n+1` 次总操作（含一次额外“停”操作）中选择 `2^n - k` 个位置放置向下操作，即 `C(n+1, 2^n - k)`。
    ///
    /// 算法：枚举 `n = 0,1,2,...` 直到 `2^n - n - 1 > k`，对每个满足条件的 `n` 累加组合数 `C(n+1, 2^n - k)`。
    pub fn ways_to_reach_stair(k: i32) -> i32 {
        (0..)                                 // 无限枚举 n
            .map(|n| (n, 1 << n))            // (n, 2^n)
            .map(|(n, npow)| (n, npow, npow - n - 1)) // 添加 lower bound
            .take_while(|&(_, _, lower)| lower <= k) // lower > k 后不可能再满足
            .filter(|&(_, npow, lower)| lower <= k && k <= npow) // 目标在区间内
            .map(|(n, npow, _)| Self::comb(n + 1, npow - k)) // 计算组合数
            .sum()                            // 累加所有方案数
    }

    /// 计算组合数 C(n, k)（辅助函数）
    /// 使用递推公式：C(n, k) = C(n, k-1) * (n - k + 1) / k，迭代计算。
    /// 参数均为非负整数，结果保证在 i32 范围内。
    fn comb(n: i32, k: i32) -> i32 {
        if k < 0 || k > n {
            return 0;
        }
        let k = k.min(n - k);          // 利用对称性，C(n, k) = C(n, n-k)
        (0..k)
            .fold(1i64, |acc, i| {
                // 第 i 步（i 从 0 开始）：分子乘 (n - i)，分母除 (i + 1)
                acc * (n - i) as i64 / (i + 1) as i64
            }) as i32
    }
}
```
