## 连续数列

给定一个整数数组，找出总和最大的连续数列，并返回总和。

示例：

输入： [-2,1,-3,4,-1,2,1,-5,4]
输出： 6
解释： 连续子数组 [4,-1,2,1] 的和最大，为 6。
进阶：

如果你已经实现复杂度为 O(n) 的解法，尝试使用更为精妙的分治法求解。

```
use std::cmp::max;  // 导入标准库的max函数，用于比较两个值并返回较大值

impl Solution {  // 为Solution结构体实现方法（LeetCode要求的入口）
  pub fn max_sub_array(nums: Vec<i32>) -> i32 {  // 公共方法：接受整数向量，返回最大子数组和
    let mut max_v = nums[0];  // 初始化全局最大值为数组第一个元素（数组至少有一个元素）
    // 使用fold迭代器方法，初始累加器为0，遍历每个元素
    // sum: 当前累加的子数组和，x: 当前元素的引用
    nums.iter().fold(0, |sum, x| {
      let mut ret = max(sum, 0);  // 如果当前累加和为负数，则重置为0（Kadane算法核心）
      ret += *x;  // 将当前元素加到重置后的累加和上（解引用x获取值）
      max_v = max(max_v, ret);  // 更新全局最大值：比较当前最大值和新子数组和
      ret  // 返回新的累加和，作为下一次迭代的sum参数
    });
    max_v  // 返回全局最大值
  }
}
```
