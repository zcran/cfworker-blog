## 商品折扣后的最终价格

给你一个数组 prices ，其中 prices[i] 是商店里第 i 件商品的价格。

商店里正在进行促销活动，如果你要买第 i 件商品，那么你可以得到与 prices[j] 相等的折扣，其中 j 是满足 j > i 且 prices[j] <= prices[i] 的 最小下标 ，如果没有满足条件的 j ，你将没有任何折扣。

请你返回一个数组，数组中第 i 个元素是折扣后你购买商品 i 最终需要支付的价格。

```
impl Solution {
    /// 计算商品折扣后的最终价格
    /// 规则：对于每个商品，找到它右边第一个价格 <= 它的商品，享受差价折扣
    ///
    /// # 算法
    /// 单调栈（递增栈）+ 反向遍历
    ///
    /// # 时间复杂度 O(n)  空间复杂度 O(n)
    pub fn final_prices(prices: Vec<i32>) -> Vec<i32> {
        let mut ans = vec![0; prices.len()];  // 存储最终价格
        let mut st = vec![0];  // 单调递增栈（存储可能成为折扣值的价格），哨兵 0 保证栈永不为空

        // 反向遍历：从右向左找右边第一个 <= 当前值的元素
        for (i, p) in prices.into_iter().enumerate().rev() {
            // 维护栈的递增性：弹出所有大于当前价格的栈顶元素
            // 因为当前价格 p 更小，对左侧元素来说它是更优的折扣候选
            while *st.last().unwrap() > p {
                st.pop();  // 比 p 大的价格永远不会成为左侧更小数的折扣，可以丢弃
            }

            // 栈顶元素就是右边第一个 <= p 的价格（折扣值）
            // 如果没有折扣，栈顶是哨兵 0，折扣为 0
            ans[i] = p - st.last().unwrap();

            // 当前价格入栈，作为左侧元素的潜在折扣
            st.push(p);
        }

        ans
    }
}
```
