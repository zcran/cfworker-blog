## 平衡装运的最大数量

给你一个长度为 n 的整数数组 weight，表示按直线排列的 n 个包裹的重量。装运 定义为包裹的一个连续子数组。如果一个装运满足以下条件，则称其为 平衡装运：最后一个包裹的重量 严格小于 该装运中所有包裹中 最大重量 。

选择若干个 不重叠 的连续平衡装运，并满足 每个包裹最多出现在一次装运中（部分包裹可以不被装运）。

返回 可以形成的平衡装运的最大数量 。

```
impl Solution {
    /// 返回可以形成的平衡装运的最大数量
    ///
    /// 平衡装运条件：最后一个包裹重量 < 该装运中所有包裹的最大重量
    /// 贪心策略：每找到相邻递减对 `(i-1) > (i)` 就形成一个长度为2的装运
    pub fn max_balanced_shipments(weight: Vec<i32>) -> i32 {
        let n = weight.len();
        let mut ans = 0;
        let mut i = 1;

        while i < n {
            if weight[i - 1] > weight[i] {
                ans += 1;
                i += 2;  // 跳过当前两个元素，保证不重叠
            } else {
                i += 1;
            }
        }

        ans
    }
}
```
