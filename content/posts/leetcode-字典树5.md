## 字典序排数

给你一个整数 n ，按字典序返回范围 [1, n] 内所有整数。

你必须设计一个时间复杂度为 O(n) 且使用 O(1) 额外空间的算法。


```
impl Solution {
    /// 生成从 1 到 n 的字典序排列
    ///
    /// 字典序（lexicographical order）也称为"词典顺序"
    /// 例如：1, 10, 11, 12, ..., 19, 100, 101, ..., 2, 20, 21, ...
    ///
    /// # 参数
    /// - `n`: 最大数字（包含）
    ///
    /// # 返回
    /// 按字典序排列的 1..=n 的所有数字
    ///
    /// # 示例
    /// ```
    /// let result = Solution::lexical_order(13);
    /// assert_eq!(result, vec![1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9]);
    /// ```
    ///
    /// # 算法核心
    /// 模拟深度优先遍历（DFS）一棵 10 叉树：
    /// ```
    ///          1        2    3    ...    9
    ///        / | \
    ///      10 11 12 ...
    ///      /
    ///    100 ...
    /// ```
    pub fn lexical_order(n: i32) -> Vec<i32> {
        // 预分配固定大小的数组，避免动态扩容
        let mut ans = vec![0; n as usize];
        let mut num = 1;  // 当前数字，从 1 开始

        // 生成 n 个数字
        for i in 0..n as usize {
            ans[i] = num;  // 将当前数字存入结果

            // 情况 1: 可以向下深入（添加后缀 0）
            // 例如: 1 -> 10, 10 -> 100
            if num * 10 <= n {
                num *= 10;  // 向下走（深度优先）
            }
            // 情况 2: 无法向下，需要向右或向上回溯
            else {
                // 向右移动时，需要跳过以 9 结尾的数字
                // 或者当数字已经达到 n 时，需要回溯到上一层
                while num % 10 == 9 || num + 1 > n {
                    num /= 10;  // 向上回溯（移除最后一位）
                }
                num += 1;  // 向右移动（同一层下一个数字）
            }
        }

        ans
    }
}
```
