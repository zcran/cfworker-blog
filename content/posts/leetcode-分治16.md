## 特殊的二进制字符串


特殊的二进制字符串 是具有以下两个性质的二进制序列：

· 0 的数量与 1 的数量相等。
· 二进制序列的每一个前缀码中 1 的数量要大于等于 0 的数量。

给定一个特殊的二进制字符串 s。

一次移动操作包括选择字符串 s 中的两个连续的、非空的、特殊子串，并交换它们。两个字符串是连续的，如果第一个字符串的最后一个字符与第二个字符串的第一个字符的索引相差正好为 1。

返回在字符串上应用任意次操作后可能得到的字典序最大的字符串。



```
impl Solution {
    /// 构造最大的特殊二进制字符串
    ///
    /// 特殊二进制字符串定义：
    /// 1. 只由 '0' 和 '1' 组成
    /// 2. 任意前缀中 '1' 的个数 >= '0' 的个数
    /// 3. 整个字符串中 '1' 和 '0' 的数量相等
    ///
    /// 算法思路：
    /// 1. 将字符串按"平衡段"拆分（前缀和归零的位置）
    /// 2. 每个平衡段内部递归处理，去掉首尾的 '1' 和 '0' 后递归
    /// 3. 递归结果为 "1" + 内部递归结果 + "0"
    /// 4. 将所有平衡段的结果按字典序降序排列后拼接
    pub fn make_largest_special(s: String) -> String {
        let mut cnt = 0;  // 计数器：遇 '1' 加1，遇 '0' 减1

        // 1. 按平衡段拆分字符串（当 cnt == 0 时切分）
        let mut arr = s
            .split_inclusive(|ch| {
                cnt += if ch == '1' { 1 } else { -1 };
                cnt == 0  // 当计数器归零时，表示找到一个完整的平衡段
            })
            .map(|curr| {
                // 对每个平衡段：
                // - 去掉首尾的 '1' 和 '0'（curr[0] = '1', curr[-1] = '0'）
                // - 递归处理内部子串
                // - 重新包装为 "1" + 内部结果 + "0"
                let inner = &curr[1..curr.len() - 1];
                format!("1{}0", Self::make_largest_special(inner.to_string()))
            })
            .collect::<Vec<_>>();

        // 2. 将各段按字典序降序排列（最大排列）
        arr.sort_by(|a, b| b.cmp(a));

        // 3. 拼接所有段
        arr.concat()
    }
}
```
