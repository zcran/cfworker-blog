## 对字母串可执行的最大删除数

给你一个仅由小写英文字母组成的字符串 s 。在一步操作中，你可以：

删除 整个字符串 s ，或者
对于满足 1 <= i <= s.length / 2 的任意 i ，如果 s 中的 前 i 个字母和接下来的 i 个字母 相等 ，删除 前 i 个字母。
例如，如果 s = "ababc" ，那么在一步操作中，你可以删除 s 的前两个字母得到 "abc" ，因为 s 的前两个字母和接下来的两个字母都等于 "ab" 。

返回删除 s 所需的最大操作数。

```
impl Solution {
    /// 计算将字符串 s 全部删除所需的最大操作次数。
    /// 每次操作可以删除一个前缀，前提是该前缀等于紧随其后的相同长度的子串。
    /// 使用 LCP（最长公共前缀）数组 + 动态规划，时间复杂度 O(n²)，空间复杂度 O(n²)。
    pub fn delete_string(s: String) -> i32 {
        let s = s.as_bytes();
        let n = s.len();

        // 特判全部字符相同的情况：可以每次删除一个字符，共 n 次
        if s.iter().skip(1).all(|&c| c == s[0]) {
            return n as i32;
        }

        // ---------- 1. 计算 LCP 数组 ----------
        // lcp[i][j] 表示 s[i..] 和 s[j..] 的最长公共前缀长度（i < j）
        // 使用二维 Vec 存储，索引范围 0..=n
        let mut lcp = vec![vec![0; n + 1]; n + 1];
        // 从后向前递推，利用 lcp[i][j] = lcp[i+1][j+1] + 1（如果 s[i] == s[j]）
        for i in (0..n).rev() {
            for j in (i + 1..n).rev() {
                if s[i] == s[j] {
                    lcp[i][j] = lcp[i + 1][j + 1] + 1;
                }
            }
        }

        // ---------- 2. 动态规划 ----------
        // f[i] 表示从 i 开始到末尾的子串最多可以删除的次数
        let mut f = vec![0; n];
        for i in (0..n).rev() {
            // 枚举可能的删除长度 j（前缀长度）
            // 条件：i + 2*j <= n，确保存在第二个相同子串
            for j in 1..=((n - i) / 2) {
                // 如果 s[i..i+j] 与 s[i+j..i+2j] 相等，则可以从 i 处删除一次后跳到 i+j
                if lcp[i][i + j] >= j {
                    f[i] = f[i].max(f[i + j]);
                }
            }
            // 加上当前这一次删除
            f[i] += 1;
        }

        f[0] as i32
    }
}
```
