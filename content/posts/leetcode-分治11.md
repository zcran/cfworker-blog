## 摆动排序 II

给你一个整数数组 nums，将它重新排列成 nums[0] < nums[1] > nums[2] < nums[3]... 的顺序。

你可以假设所有输入数组都可以得到满足题目要求的结果。

```
impl Solution {
    /// 将数组重排为摆动排序：nums[0] <= nums[1] >= nums[2] <= nums[3] >= ...
    ///
    /// 算法思路：
    /// 1. 克隆原数组并对克隆体进行排序
    /// 2. 将较小的一半元素依次放入原数组的偶数下标（从中间向下取）
    /// 3. 将较大的一半元素依次放入原数组的奇数下标（从末尾向前取）
    pub fn wiggle_sort(nums: &mut Vec<i32>) {
        // 第1步：创建原数组的排序副本
        let mut 排序副本 = nums.clone();
        排序副本.sort();

        let 长度 = nums.len();

        // 从排序副本中取数的两个指针：
        // - p 指向较小一半的最后一个元素（中间位置）
        // - q 指向较大一半的最后一个元素（末尾位置）
        let mut p = (长度 - 1) / 2;  // 较小半区的末尾索引
        let mut q = 长度 - 1;        // 较大半区的末尾索引

        // 第2步和第3步：按摆动顺序填充原数组
        for i in 0..长度 {
            if i % 2 == 1 {
                // 奇数下标：从较大半区取值（右侧）
                nums[i] = 排序副本[q];
                q -= 1;
            } else {
                // 偶数下标：从较小半区取值（左侧）
                nums[i] = 排序副本[p];
                p -= 1;
            }
        }
    }
}
```
