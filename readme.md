##### wrangler 命令

支持 Wrangler 将脚本部署到 Cloudflare，您需要登录到 Cloudflare 帐户完成身份验证。
wrangler login

从此 Worker 初始化新项目
wrangler init --from-dash worker-solitary-credit-db82


将您的项目部署到 Cloudflare 的全球网络：
wrangler publish
